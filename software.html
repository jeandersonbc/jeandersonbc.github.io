Software

SYMBOLRT is a model-based test generation tool developed to automatically generate test cases for real-time systems. In this project, I improved the user experience by implementing a specification language for Real-Time systems and a compiler. This award-winning project was developed at Software Practices Lab (SPLab). Access the official website for more information.

DesignWizard is an API that supports the automated inspection of Java programs. Wondering how is the design of your project? What if you could invoke magic methods to retrieve classes and the relation between them? Would be important to know if class A inherits class B but not class C? DesignWizard is particularly useful to support automated code reviews and to write design tests in an easy and clean way. No special formal language or notation is required to describe design rules with DesignWizard, just plain Java and JUnit. This award-winning API was developed by Joao Arthur Brunet and I help him to maintain the code. Access the official website for more information.

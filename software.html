Software

SYMBOLRT is a model-based test generation tool developed to automatically generate test cases for real-time systems. In this project, I improved the user experience by implementing a specification language for Real-Time systems and a compiler. This award-winning project was developed at Software Practices Lab (SPLab). Access the official website for more information.

DesignWizard is an API that supports the automated inspection of Java programs. DesignWizard is particularly useful to support automated code reviews and to write design tests in an easy and clean way. No special formal language or notation is required to describe design rules with DesignWizard, just plain Java and JUnit. This award-winning API was developed by Joao Arthur Brunet. Nowadays, I'm a contributor of the project and I help Joao to maintain and evolve it. Access the official website for more information.
